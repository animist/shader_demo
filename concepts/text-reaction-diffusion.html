文字が育つ反応拡散

smapho_demo / 案 #3 — 既存の reaction-diffusion デモを文学テクストで種付けする

ライブシミュレーション · Gray-Scott t = 00:00

「羅生門」冒頭の段落を 3 本のトラック（右・中・左）に並べ、各トラックが独立した 静止 → 立ち上がり → 拡散 → 再集結 サイクルを 7 秒ずつズラして 回す。 あるトラックが読める姿で立っているとき、別のトラックは模様に解け、また別のトラックは 模様から次の行を集めている。読み終わる前に模様になり、隣の模様から次の言葉が再び立ち上がる。 13 行で一段落、終端で 1 行目へループ。

進行

各トラックは独立な 21 秒サイクル。sim は常時全速力で回し、位相は target に向かう lock 強度（hold=1.0 / ramp=1−p³ / dissolve=0 / reform=1−(1−p)³） でだけ表現する。初期の hold 延長で 7 秒ずつズラし、定常状態に入る。

  1. 2 s

    静止 — 行を読む（lock = 1）

    毎フレーム target の値で完全に上書き。sim はトラックの外側で進むが、このトラックは text のまま留まる。

  2. 2 s

    立ち上がり（lock = 1−p³）

    lock が cubic ease-in で 1 → 0 に解放される。最初は固く保たれ、終盤で急に sim に明け渡される。画数の細い部分から先に変質する。

  3. 14 s

    拡散（lock = 0）

    このトラックは sim 任せ。プリセットの f, k に従って固有の崩れ方をする。mitosis なら胞子状、coral なら細胞状、spots なら孤立点、chaos なら不規則な滲み。

  4. 3 s

    再集結 — 次の行（lock = 1−(1−p)³）

    そのトラックの x-range に 次の行のグリフを target として焼き込み、lock を cubic ease-out で 0 → 1 に上げる。sim が churning する中、新しい文字が引き寄せられて立ち上がる。終端で target に snap。

  5. ↻

    3 トラック並列 / 7 秒ズレ / global counter

    初期 hold を 0s / 7s / 14s で延長して定常状態へ。reform を抜ける順に nextGlobalLine から行番号を払い出すので、読み順 0→1→2→3→… が保持される。最終行の reform 完了でループ。

素材

或日の暮方の事である。一人の下人が、羅生門の下で雨やみを待つてゐた。 芥川龍之介「羅生門」 (1915) · 青空文庫 No.879 · 旧字旧仮名

青空文庫は CORS を許可していないため fetch は不可。本文 txt 版（パブリックドメイン）を repo に同梱する形にする。 ルビ記法 漢字《かんじ》 と注記 ［＃...］ は10行程度のパーサーで剥がせる。短編1本で数十KB。

実装

既存の demos/reaction-diffusion.html をベースに、以下を足すだけで成立する。Ping-Pong レンダリングと自動種まきの足回りは既に書いてある。

  • text-to-texture パス: Canvas2D で本文を縦書きレンダ → R32F テクスチャ化 → V チャンネルの初期値として転送。
  • seed inject パス: ⚙ パネルの「再シード」ボタンで、現在の V に新しい文章を加算注入。読み手の意思で「次の段落」「次の作品」を蒔ける。
  • 挙動プリセット: mitosis / coral / spots / solitons の f,k ペア切替。文字の崩れ方が劇的に変わる（上のキャンバスでも試せます）。
  • paramPanel 追加項目: text:作品ID, range:フォントサイズ, button:再シード, button:次の節へ。値は URL ハッシュへ自動保存される（既存仕様）。
  • カラーマップ: 既存 reaction-diffusion デモは emerald → purple。本案は素材に合わせて苔の闇から螢光の青磁へ。朽葉色 #b5573a を縁取りアクセントに。
このページの canvas について ／ 実際に Gray-Scott を走らせている（200×360 グリッド × 3 ステップ/フレーム、Canvas2D + Float32Array）。 本番のデモは WebGL2 の半精度 RT で同等以上の解像度を出します。 日本語縦書きのグリフは fillText でカラム配置・右起こし。
